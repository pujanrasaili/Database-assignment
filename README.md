# Database Assignment – BankDB

This repository contains the complete SQL assignment for DBMS, including:

## Features

- **Accounts Table**: Demonstrates transactions, COMMIT, ROLLBACK, and SAVEPOINT.  
- **Employees Table**: Includes triggers for BEFORE INSERT and AFTER UPDATE.  
- **Salary Log Table**: Logs salary changes automatically.  
- **Stored Procedures**:  
  1. `getEmployees()` – Retrieve all employees  
  2. `addEmployee()` – Insert a new employee  
  3. `updateEmployeeSalary()` – Update salary for a specific employee  
  4. `transferMoney()` – Transfer money between accounts using transactions  

## Tools Used

- MySQL  
- MySQL Workbench
